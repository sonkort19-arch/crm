Файлы бренд-графики Too CRM
===========================

logo.svg — логотип для светлой темы (чёрное яблоко со швом, прозрачный фон).

logo-dark.svg — логотип для тёмной темы (зелёное яблоко со швом, прозрачный фон).

favicon.svg — SVG-иконка вкладки браузера.

favicon-32.png — PNG 32×32 для совместимости браузеров.

apple-touch-icon.png — PNG 180×180 для ярлыка на мобильных устройствах.

logo-512.png — PNG 512×512 для крупных системных и маркетинговых задач.
